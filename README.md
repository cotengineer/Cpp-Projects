Individual source file projects made with C++

1) Animation: Customized smooth animation of objects made in Linux environment; REQUIRES graphics.h library to function!!!
    example compilation and run: 
        $ g++ -o filename filename.cpp -lgraph
        $ ./filename
        
2) Tic Tac Toe Game: 2 play terminal GUI for competitive tic tac toe

3) 2D Robot Simulation: Outputs X- Y- graph coordinates of robot from given string. While loop can be added to continuously receive and execute moves. Recalling function currently gives relative direction from robot, not global.
